# Building with Parsley

Parsley is a **minimalist language** for generating HTML/XML with first-class file I/O.

## Features

- Simple syntax
- Built-in file handling
- Component-based templating

## Code Example

```parsley
let name = "World"
<h1>Hello {name}</h1>
```
